# Brokerage IT Ticket Tracker

A personal help desk tracking application for real estate agent support, follow-up, and reporting.

## Quick Start

### Online
Open the HTML file directly in your browser:
- Click "Code" → "Download ZIP" if you cloned this repo
- Or visit: [bailiemeighan.github.io/help/brokerage-helpdesk-tracker.html](https://bailiemeighan.github.io/help/brokerage-helpdesk-tracker.html)

### Offline
1. Download `brokerage-helpdesk-tracker.html`
2. Double-click to open in your browser
3. Bookmark the page for quick access

## Features

- **Ticket Management**: Create, edit, and track support requests
- **Filtering & Search**: Find tickets by agent, status, priority, category, and more
- **Reporting**: View statistics by category, status, and office
- **Time Tracking**: Log hours spent on each ticket
- **Data Export**: Export to CSV for use in spreadsheets
- **Backup & Import**: Export JSON backup and import on another computer
- **Print Ready**: Generate printable reports

## Data Storage

Tickets are stored locally in your browser. To move tickets between computers:

1. **On the computer with your tickets:**
   - Open the app
   - Click "Backup JSON"
   - Save the file somewhere safe (OneDrive, Dropbox, external drive, etc.)

2. **On the new computer:**
   - Open the app
   - Click "Import JSON"
   - Select the backup file you saved

## Recommended Habits

- Backup your JSON at the end of each day
- Use the Sample Data button to see how the tracker works
- Export monthly reports for your records

## Need Help?

The app is self-contained in a single HTML file. No internet required to use it.
